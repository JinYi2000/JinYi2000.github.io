浅拷贝

如果有属性是对象的话拷贝的是地址

  1. Object.assign

    var b = Object.assign({},a);

  2. 点展开符

    var b = {...a};

深拷贝

属性是对象的话拷贝的不是地址

  1. JSON.stringify. JSON.parse

    var b = JSON.parse(JSON.stringify(a));

  2. 递归拷贝
