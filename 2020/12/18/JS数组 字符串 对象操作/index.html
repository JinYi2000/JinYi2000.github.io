    Slice

    返回由begin和end决定的原数组的浅拷贝，原数组不会被改变

    slice

    改变原数组

    参数： 开始位置 要删除的元素的个数 要添加的元素…

    返回值：删除掉的元素

    indexOf

    every some

    Every:测试一个数组内的所有元素是否都能通过某个函数的测试

    Some:有通过的就行

    filter map forEach

    Filter 创建一个新数组, 其包含通过所提供函数实现的测试的所有元素

    map 创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。(不改变原数组)

    forEach 对数组的每个元素执行一次给定的函数
